Worked on the autonomous navigation and vehicle movement subsystem of an emergency response vehicle. Responsible for GPS integration and real-time location tracking, vehicle motion control, and embedded hardware integration. Implemented movement control using the PicUno RP2040 and integrated the ESP32 XIAO C3 for GPS and MPU6050-based orientation data processing. Worked on A*-based shortest-path navigation and integrated ultrasonic sensors for obstacle detection during autonomous movement. Contributed to sensor interfacing, motor control, embedded logic, and testing of the vehicle’s autonomous navigation system.
